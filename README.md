# 🌐 Sree Meghana — Personal Portfolio Website

A responsive personal portfolio website built with HTML, CSS, and JavaScript, showcasing my education, skills, projects, and contact information.

---

## 📌 Live Sections

| Section | Description |
|---|---|
| **Home** | Introduction with social media links |
| **About** | Brief personal bio |
| **Education** | Academic background & internships |
| **Skills** | Coding and professional skill bars |
| **Projects** | Featured personal and academic projects |
| **Contact** | Contact form for inquiries |

---

## 🛠️ Tech Stack

- **HTML5** — Page structure and semantics
- **CSS3** — Styling, animations, and responsive layout
- **JavaScript** — Scroll animations, sticky header, mobile menu toggle
- **Boxicons** — Icon library (`unpkg.com/boxicons`)

---

## 📁 Project Structure

```
portfolio/
├── index.html      # Main HTML file
├── style.css       # Stylesheet with animations and responsive design
└── script.js       # JS for navbar toggle and scroll behavior
```

---

## ✨ Features

- Smooth scroll-based animations using CSS custom properties (`--i`)
- Sticky header on scroll
- Responsive hamburger menu for mobile devices
- Active navigation link highlighting based on scroll position
- Animated skill progress bars
- Contact form UI
